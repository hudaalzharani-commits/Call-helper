## 🧮 **كيفية حساب النسب:**

### **Formula الأساسية:**

```typescript
// 1. Base Score
baseScore = 30 (لأي وصف >= 10 أحرف)

// 2. Positive Keywords
POSITIVE_KEYWORDS.forEach(pattern => {
  matches = pattern.keywords.filter(keyword => description.includes(keyword))
  score += matches.length * pattern.weight
})

// 3. Negative Keywords
NEGATIVE_KEYWORDS.forEach(keyword => {
  if (description.includes(keyword)) {
    score -= 8
  }
})

// 4. Length Bonus
if (wordCount > 15) score += 10
else if (wordCount > 8) score += 5

// 5. Final Score
finalScore = Math.max(0, Math.min(100, score))
```

### **Positive Keywords بالأوزان:**

| الكلمات | الوزن |
|---------|-------|
| تعطل، عطل، خطأ، مشكلة، لا يعمل، توقف | 15 |
| نظام، برنامج، تطبيق، موقع، منصة | 12 |
| دفع، مالي، فاتورة، رسوم، تحويل، استرداد | 12 |
| تشغيلي، عملية، إجراء، خدمة، معالجة | 10 |
| شكوى، اعتراض، استياء، غير راضي | 10 |
| استفسار، سؤال، معلومات، توضيح | 8 |

### **Negative Keywords (كل واحدة -8):**

شيء، شي، حاجة، مدري، ما أدري، تقريبا، ممكن، يمكن، شوف، شف، كذا، مو متأكد، مش متأكد

**الوصف:**
```

```

**الحساب:**
```typescript

// Positive Keywords:
"يواجه" ← لا يطابق

"إتمام" ← لا يطابق
"عملية" ← 10 (من مجموعة: تشغيلي، عملية، إجراء...)
"دفع" ← 12 (من مجموعة: دفع، مالي، فاتورة...)
"إلكتروني" ← لا يطابق
"نظام" ← 12 (من مجموعة: نظام، برنامج، تطبيق...)

totalPositive = 15 + 10 + 12 + 12 = 49

// Negative Keywords:
لا يوجد → 0

// Length Bonus:
wordCount = 11 (> 8) → +5

// Final:
score = 30 + 49 + 5 = 84

// Description Matching:
يطابق rp4 بنسبة ~75% → يستخدم الرد المسجل

```

**النتيجة:**
- **confidenceScore**: 84%
- **descriptionMatchPercentage**: 75%
- **النسبة المعروضة**: 84% (دقة ممتازة - 🟢 Green)
- **الأزرار**: زر "أفدتك؟" فقط ✅
- **الرد**: رد مسجل من قاعدة البيانات
